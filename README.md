# BlogMob | The Blogging App

</div>

<div align="left">
<br>

BlogMob is a blogging app where users can post blogs and like the blogs they love. <br>

<br>
<hr>
<br>

## Features⚡️

- Create customised profiles
- Read others' blog in Blogfeed
- Write blogs
- Like blogs you love
